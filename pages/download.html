PyBrain on GitHub

We use git and GitHub to manage our code. If you want to clone the PyBrain repository to get the latest version of PyBrain, you can do so by using this command (you need to have git installed on your system): 

git clone git://github.com/pybrain/pybrain.git

This will clone the repository to your local machine. Please note, that you cannot commit any changes to the repository with this address. If you want to take active part in PyBrain development, please get in contact with us.

You can also download a zipped version of the current master branch from GitHub.
