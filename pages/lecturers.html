PyBrain for Lecturers

One of the main design goals for PyBrain is for it to be a library that is suited to be used in the context of a machine learning course. So far it has only been used for courses at the Technische Universität München, but you are welcome to use it for your courses as well. Almost all the basic algorithms and principles of machine learning are implemented and well-documented. PyBrain can be used, for example, for small course assignments or larger student projects. More advanced courses are supported as well, by a large number of implementations of state-of-the-art algorithms, some at the bleeding edge of research.

If you use PyBrain for your course and write extensions or interesting examples using it, we'd appreciate if you could contribute them to the project via the Developer Group so that they can be distributed with future versions.
